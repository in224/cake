# js-Rock Paper Scissors
자바스크립트로 코딩한 가위바위보

## 기능목록

  - [x] 결과
  + 가위바위보 결과 메세지가 출력된다.
 ---
  - [x] 이긴 횟수 카운트
  + 가위바위보 결과 중 이긴 횟수를 카운트한다.
 ---
  - [x] 가위바위보 이미지 출력
  + 나는 아래 버튼에서 제출할 가위, 바위, 보를 고르면 해당 이미지가 출력된다.
  + 컴퓨터는 랜덤으로 가위, 바위, 보가 출력된다.
  + 컴퓨터의 가위바위보 확률을 조작할 수 있다.
  ---
  - [x] 가위바위보 버튼
  + 가위, 바위, 보 3가지 버튼 중 한가지를 선택하면 해당하는 이미지가 출력된다.
